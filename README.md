# Сайт компании — образовательное оборудование для школ и детских садов

## О проекте

Компания поставляет оборудование (интерактивные панели, конструкторы, робототехника, настенные панели) в детские сады и школы. Сайт решает две задачи:
- Привлекает директоров и заведующих полезными ресурсами (ФГОС, ФОП, гранты, инструменты)
- Рекламирует товары компании и генерирует заявки

---

## Технологии

- Чистый HTML / CSS / JavaScript (без фреймворков)
- Адаптивный дизайн (мобильные + десктоп)
- Шрифты: Google Fonts (Roboto) — CDN
- Иконки: Font Awesome — CDN
- Генерация документов: **docx.js** (CDN) — создание .docx прямо в браузере
- Отправка форм: **EmailJS** — бесплатный сервис, не нужен backend
- Аналитика: **Яндекс.Метрика**
- Сохранение состояния чеклистов: **localStorage**
- Контент (новости, гранты, товары): **JSON-файлы** — редактируются без правки HTML

## Разделы сайта

### 1. Главная (index.html)
- Шапка с логотипом, навигацией, телефоном и кнопкой Telegram/WhatsApp
- Hero-блок: "Всё для современной школы и детского сада"
- Карточки-ссылки на ключевые разделы
- Блок "О компании" + сертификаты/лицензии на товары

- Подвал: реквизиты, ссылка на политику конфиденциальности

### 3. Документы (documents.html)
- Данные загружаются из `data/documents.json` — обновление без правки HTML
- Карточки по категориям: ФГОС, ФОП, методические материалы
- Ссылки на официальные источники (проверяются при обновлении JSON)
- Фильтр по типу документа

### 3. Гранты и федеральные проекты (grants.html)
- Данные из `data/grants.json`
- Карточки: название, сумма, дедлайн, статус (активный / завершён)
- Фильтр: активные / завершённые / федеральные проекты
- Актуальные источники грантов:
  - Президентский фонд культурных инициатив
  - Фонд гуманитарных проектов
  - Региональные гранты (по субъектам РФ)
  - Минпросвещения (edu.gov.ru/national-project)
- Блок "Под какой грант подходит наше оборудование" → ссылка на каталог
- При обновлении — редактировать только JSON, не HTML

### 4. Каталог товаров (catalog.html)
- Данные из `data/catalog.json`
- Карточки: фото, название, описание, **цена или "Цена по запросу"**
- Категории: интерактивные панели, конструкторы, робототехника, настенные панели
- Ссылки на сертификаты и документы на товар
- Кнопка "Оставить заявку" → форма → EmailJS отправляет на email

### 5. Новости (news.html)
- Данные из `data/news.json`
- Новостные карточки с датой и описанием
- Добавление новости = добавить запись в JSON

### 6. Генератор документов (generator.html)
- Реализован через **docx.js** (браузерная библиотека)
- Генератор приказов: форма → готовый .docx для скачивания
- Шаблоны рабочих программ: предмет, класс, часы → структура документа
- Генератор заявок на гранты
- Акты приёма оборудования

### 7. Календарь дедлайнов (deadlines.html)
- Данные из `data/grants.json` (дедлайны берутся оттуда же)
- Таблица: сдача отчётов, подача грантов, аттестации, ГИА, ЕГЭ
- Фильтр по типу и месяцу
- Визуальное выделение ближайших дат

### 8. Чеклисты для проверок (checklists.html)
- Интерактивные списки: Рособрнадзор, пожарная безопасность, СанПиН
- Прогресс-бар готовности
- **Прогресс сохраняется в localStorage** — не сбрасывается при перезагрузке
- Кнопка "Распечатать чеклист"

### 9. Библиотека шаблонов (templates.html)
- Файлы хранятся в `downloads/templates/`
- Категории: трудовые договоры, положения, программы, акты
- Поиск по названию (JS-фильтрация)

### 10. Оснащение по ФГОС (fgos-equipment.html)
- Что обязан иметь детсад/школа по стандарту
- Привязка к товарам: "Этот пункт закрывает → [товар из каталога]"
- Подбор комплектации под бюджет гранта
- Истории успеха школ

### 11. Контакты (contacts.html)
- Телефон, email, Telegram, WhatsApp
- Юридический адрес, ИНН, ОГРН — для проверки контрагента
- Форма обратного звонка → EmailJS
- Режим работы

### 12. Политика конфиденциальности (policy.html)
- Обязательна по 152-ФЗ при сборе персональных данных через форму
- Стандартный шаблон с данными компании

## Форма заявки — настройка EmailJS

1. Зарегистрироваться на emailjs.com (бесплатно до 200 писем/месяц)
2. Создать Email Service (Gmail или любой SMTP)
3. Создать Email Template
4. Вставить `serviceID`, `templateID`, `publicKey` в `js/form.js`

## SEO

- Уникальные `<title>` и `<meta name="description">` на каждой странице
- `robots.txt` — разрешает индексацию всех страниц
- `sitemap.xml` — перечень всех URL для Яндекса и Google
- Яндекс.Метрика — счётчик вставляется в `<head>` каждой страницы

## Обновление контента

| Что обновить | Как |
|---|---|
| Новость | Добавить запись в `data/news.json` |
| Грант / дедлайн | Добавить/изменить в `data/grants.json` |
| Товар в каталоге | Добавить/изменить в `data/catalog.json` |
| Документ (ФГОС) | Добавить/изменить в `data/documents.json` |
| Шаблон для скачивания | Положить .docx в `downloads/templates/` |
